PC 켤 때, 나만의 업무 환경 자동 세팅하기

매일 반복되는 PC 환경 설정, 이제 클릭 한 번으로 완벽하게 자동화하세요.

CHAPTER 1. 기본 단계

크롬만으로 어제의 작업 환경 복원하기

목표: 외부 프로그램 없이, PC를 껐다 켜도 이전에 작업하던 크롬 창과 탭들이 그대로 복원되도록 설정합니다.

1. 크롬 '시작 그룹' 설정 변경

크롬이 마지막으로 작업하던 상태를 기억하게 만듭니다.

  1. 크롬 실행 후, 우측 상단 점 세 개(⋮) 메뉴 → '설정'으로 이동합니다.
  2. 왼쪽 메뉴에서 '시작 그룹'을 선택합니다.
  3. 옵션 중 '중단한 위치에서 계속하기'를 선택하세요.

2. (선택사항) 프로필별 바로 가기 생성

여러 구글 계정을 쓴다면, 각 계정별로 자동 실행되도록 설정하세요.

  1. 자동 실행할 프로필로 크롬을 실행합니다.
  2. 주소창에 chrome://settings/manageProfile 입력 후 엔터!
  3. '바탕화면 바로가기 만들기' 옵션을 활성화합니다.

3. 시작프로그램에 등록

PC가 켜질 때 크롬이 자동으로 실행되도록 만듭니다.

  1. 키보드에서 `Win` + `R` 키를 눌러 실행 창을 엽니다.
  2. shell:startup 입력 후 엔터!
  3. 열린 폴더 안에 바탕화면의 크롬 바로 가기를 복사해서 붙여넣습니다.

CHAPTER 2. 심화 단계

모든 앱을 원하는 환경으로 완벽하게 자동 배치하기

목표: 크롬뿐만 아니라 업무에 사용하는 모든 앱(To Do, Slack, 뮤직 등)을 하나의 스크립트로 제어하여, 단 1픽셀의 오차도 없이 완벽한 디지털 작업 공간을 자동으로 구축합니다.

1. 전체 창 레이아웃 저장하기

미래에 실행될 창들의 위치와 크기를 '무대 감독' 프로그램에 미리 알려줍니다.

  1. WindowexeMoveWindow.exe를 실행합니다.
  2. 자동 실행할 모든 프로그램(To Do, 크롬 등)을 띄우고, 원하는 최종 레이아웃으로 완벽하게 배치합니다.
  3. `WindowexeMoveWindow` 프로그램에서 `WinList`를 눌러 목록을 엽니다.
  4. 각 창을 선택하고 `설정추가(A)` 버튼을 눌러 위치 정보를 저장한 후, 최종 설정을 저장합니다.

2. 마스터 실행 스크립트 작성

모든 자동화 명령이 담긴 '지휘자의 악보'를 만듭니다.

아래 코드를 그대로 복사하여 메모장에 붙여넣으세요.

@echo off
:: ======================================================================
:: 1. 관리자 권한으로 실행하는 부분
:: ======================================================================
>nul 2>&1 "%SYSTEMROOT%\system32\cacls.exe" "%SYSTEMROOT%\system32\config\system"

if '%errorlevel%' NEQ '0' (
    echo 관리자 권한으로 다시 시작합니다...
    goto UACPrompt
) else ( goto gotAdmin )

:UACPrompt
    echo Set UAC = CreateObject^("Shell.Application"^) > "%temp%\getadmin.vbs"
    echo UAC.ShellExecute "%~s0", "", "", "runas", 1 >> "%temp%\getadmin.vbs"
    "%temp%\getadmin.vbs"
    exit /B

:gotAdmin
    pushd "%CD%"
    CD /D "%~dp0"
:: ======================================================================
:: 2. 실제 실행할 명령어 부분
:: ======================================================================

:: [명령 1] 창 위치 조절 프로그램 실행
start "MoveWindow" "D:\250613_WIndowsMove\WindowexeMoveWindow.exe"

:: 2초간 대기
timeout /t 2 /nobreak

:: [명령 2] Microsoft To Do 실행
start "" "shell:appsfolder\Microsoft.Todos_8wekyb3d8bbwe!App"

:: 2초간 대기
timeout /t 2 /nobreak

:: [명령 3] 유튜브 뮤직 랜덤 재생 실행
start "YTM" "C:\Program Files\Google\Chrome\Application\chrome.exe" "https://music.youtube.com/watch?v=dkWtahiXVJs&list=PLKUVbDOfnmDuzVeoBVtTroIzbuUeM4vBK"

exit

3. 시작프로그램에 스크립트 등록

작성한 '악보'가 PC 켤 때마다 자동으로 연주되도록 설정합니다.

  1. 메모장 내용을 `my_workspace.bat`와 같은 이름으로 저장합니다. (파일 형식: 모든 파일)
  2. 키보드에서 `Win` + `R` 키를 눌러 실행 창을 엽니다.
  3. shell:startup 입력 후 엔터!
  4. 기존에 넣어둔 바로 가기가 있다면 삭제하고, 새로 만든 `.bat` 파일 하나만 폴더 안으로 옮깁니다.

이제 매일 아침, 당신의 완벽한 작업 환경이 자동으로 펼쳐집니다.
